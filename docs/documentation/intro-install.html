What's this

Introduction

LOGATY is a PHP Library which can help you rapidly develop your next Multilingual Website. It can be used for CMS, Companies Site's, Personal site's, frameworks, etc.
It's based on PHP Language and contains a lot of Functions to easily make your interactive Website. .

Downloading

Installation

composer install phptricks-logaty --save



require: {
    "phptricks-logaty": "dev-master"
}
unboxing

What's included

Checkout the files that you'll see upon downloading

Once you download the Library, you can see a folder with several files and sub folders. You can read description of some important files in continue. Click on closed folder icon to open it.

  • Logaty Core folder its include Base Classes of library
    • Helpers
      • Detect
        • Browser.php trite - detect user browser language.
        • Country.php trite - detect user country language.
      • Switcher
        • Select.php trite - create language switch from <select> element.
        • Ul.php trite - create language switch as list from <ul> element.
      • Config.php Config Class - handel and get configuration values
      • Detect.php Detect Class - Detect User Language from browser or country
      • Link.php Link Class - to create correct links with selected language.
      • Switcher.php Switcher Class - create and customize Languages Switch.
    • Translate
      • Translator.php Base Class for handel translation / get translation from files.
    • App.php Logaty Application Container Class
    • start.php Initializing Logaty library .
  • Config Configuration variables (files).
    • options.php General Settings.
    • direction.php languages direction (ltr or rtl).
    • enabled.php enabled languages in your website.
    • flag.php language flag name (icon name en.png).
    • name.php Language Name ( Name in English & Natural Name ).
    • supported.php all languages supported in your website.
    • paths.php directory paths of (languages directory & flags directory).
    • code.php languages ISO-2 Codes - optional if you want to use it.
  • assets contain flags and translations
    • flags contain languages flags.
    • languages contain translations files & directories.
  • vendor composer components for classes auto-loader.
  • composer.json composer file - using composer for psr-4 auto-loader only.
