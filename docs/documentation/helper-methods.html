$logaty->enabled get Enabled Languages in website, if use it like property returns all enabled language, if use it like a function return if given language is enabled or not 

// return an array - all enabled languages
var_dump($logaty->enabled);
// return true if (arabic is enabled) and false if not.
var_dump($logaty->enabled('ar'));
$logaty->flag get Language flag, if use it like property returns all languages flags, if use it like a function return given language flag 

// return an array - all languages flags
var_dump($logaty->flag);
// return(arabic flag path)
var_dump($logaty->flag('ar'));
$logaty->direction get Language direction, if use it like property returns all languages directions, if use it like a function return given language direction 

// return an array - all languages directions
var_dump($logaty->direction);
// return(arabic direction)
var_dump($logaty->direction('ar')); #rtl
